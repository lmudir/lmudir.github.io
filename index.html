SOC Lab Portfolio

This project documents a complete SOC L2 investigation workflow across three simulated security incidents. Each case includes attacker activity, SIEM visibility, log analysis, timeline reconstruction, and incident response documentation using DFIR‑IRIS.

Cases

  • 0. Lab Setup
  • 1. Case 1 – Web Server Intrusion Attempt (Linux / Grav CMS)
  • 2. Case 2 – Windows Host Compromise Attempt
  • 3. Case 3 – Multi‑Stage Intrusion Simulation (Phishing → Execution → C2 → Exfiltration)

Each case contains: evidence, SIEM queries, log artifacts, MITRE ATT&CK mapping, and conclusions based on the observed activity.

Skills Demonstrated

  • Log analysis (Windows Event Logs, Sysmon, Nginx access/error logs)
  • SIEM investigation and alert correlation (Wazuh)
  • Threat detection and pattern recognition
  • Incident documentation and timeline reconstruction (DFIR‑IRIS)
  • MITRE ATT&CK technique identification and mapping
  • Basic threat hunting queries and filtering
  • Analysis of attacker behavior and TTPs

Tools & Technologies

  • Windows 10 (victim host)
